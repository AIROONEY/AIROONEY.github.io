Weekly Report

  1. 课程汇报
    上周进行生物统计学的课程汇报，差不多用了一天多的时间学习了R语言，没有学习他的基础语法就直接学他的应用还是相当的困难的，浪费了很多没必要的时间，但总体来说还算颇有收获，学习了单因素方差分析以及裂区方差分析。单因素方差分析应用的比较多，主要是用来分析某一个处理对于某一个指标的影响，判断该处理对于某一指标的影响是否显著。而对于同一处理不同组间（如不同的放牧强度）的差异性，则需要对方差分析后的结果进行duncan、tuckey、lsd多重检验来比较不同组间的区别。需要注意的是，进行单因素方差分析时，处理效应的一列，应为处理效应的大小，如：放牧强度1,4,5等等而不能是不同的处理，如氮、水添加等等。如果氮、水添加等处理被同时放到“处理”一列时，进行分析时，需要将其进行拆解，即相同的处理放在一列，而列中的数据为该处理的强度。下面是具体代码实现：
    ‘’’
    library(openxlsx)
    dat <- read.xlsx(‘F:\R statistics\DATA.xlsx’,sheet = 1,sep = ‘\t’)
    attach(dat) #可以自动匹配列表名,无需再用$符号来取列表中的某一列数据
    anova1 <- aov(Soilorganiccarbon~GI,data = dat)
    summary(anova1)
    library(agricolae)
    result <- duncan.test(anova1,’GI’)
    result
    ‘’’
    裂区方差分析，主要应用于两个处理对于同一指标的影响。如，对某一样地进行了氮的添加以及放牧，此时需要考虑两个处理对于该指标的影响，即用裂区方差分析。主要有：主效应、副效应、区组（重复），在R语言中主要运用，mcmcglmm包，具体代码如下：
    ‘’’
    library(MCMCglmm)
    m1.MCMC = MCMCglmm(Y ~ V*N, random = ~ B + B:mainplot, data=oats)#Y为因变量,V与N为两个不同的处理,random为随机效应,B为区组,mainplot为随机因子
    summary(m1.MCMC)
    ‘’’
  2. 组会汇报
    组会汇报的内容是王玲老师发表在PNAS上的多样化放牧文章，主要思想就是多样化放牧更有利于草地生态系统的多多样性以及多功能性。对于多多样性以及多功能性，王玲老师使用随机森林模型对多样性指标以及生态系统功能指标进行选取，最后确定以6个指标来表征多多样性，12个指标来表征多功能性。不同指标之间的量纲并不相同，因此王玲老师又使用0-1标准化将不同的指标划分到同一个标准下，以便进行比较。又使用多阈值法对不同处理（牛单牧、羊单牧、混牧、禁牧）对于多功能性的影响进行了分析与作图。
  3. 下周工作计划
    学习计算多功能性的几种不同方法（多阈值法、平均值法等），根据王玲老师提供的数据对数据进行一个复现。学习机器学习以及数理统计知识，完成政治PPT、生态系统生态学PPT以及人工智能现代方法的第一次作业。提升自己的代码编程能力以及数学统计思想，为后续的科研学习打下良好基础。
